# Color Scheme Documentation

## Primary Colors

### Light Mode

#### Primary (#8cb8ff)
```
RGB: 140, 184, 255
Use: Main brand color, primary buttons, key UI elements
```

#### Primary Container (#d8e6ff)
```
RGB: 216, 230, 255
Use: Highlighted sections, backgrounds for primary content
```

#### Secondary (#6b7fbd)
```
RGB: 107, 127, 189
Use: Secondary actions, complementary elements
```

#### Secondary Container (#dde1f9)
```
RGB: 221, 225, 249
Use: Secondary content backgrounds
```

#### Tertiary (#5ec4c1)
```
RGB: 94, 196, 193
Use: Accent elements, special highlights
```

#### Tertiary Container (#d0f2f0)
```
RGB: 208, 242, 240
Use: Tertiary content backgrounds
```

### Dark Mode

#### Tertiary (#86e0dd)

## Semantic Colors

### Success/Completed
```
Color: green.shade600 (#43a047)
Use: Completed surveys, success states
```

### Warning/Pending
```
Color: orange.shade600 (#fb8c00)
Use: Pending surveys, warning states
```

### Error
```
Light: #ba1a1a
Dark: #ffb4ab
Use: Error messages, validation errors
```

## Usage in Dashboard

1. **Welcome Card**: Primary to Secondary gradient
2. **New Survey Card**: Primary color border and icon
3. **View Surveys Card**: Secondary color border and icon
4. **Reports Card**: Tertiary color border and icon
5. **Patients Card**: Secondary color border and icon
6. **Completed Stat**: Green
7. **Patients Stat**: Primary
8. **Pending Stat**: Orange
9. **Total Stat**: Secondary

## Accessibility Notes

All color combinations meet WCAG 2.1 Level AA standards for contrast:
- Primary on white: 5.2:1 ratio
- Text on primary: 4.8:1 ratio
- Proper contrast maintained in dark mode
